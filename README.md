
# Personal Blog Website ✦

A beautifully responsive, micro-dashboard style personal blog subpage designed to showcase a modern, creative aesthetic. Built utilizing standard web technologies, this project features a balanced grid layout blending structured technical portfolios with expressive storytelling blocks.

## ✨ Features

- **Cohesive Pastel Aesthetic:** Uses a hand-picked, elegant color scheme balancing soft Lavender (`#c5bbf2`) and Peach (`#ffd8be`) tile surfaces against a dark background contrast.
- **Micro-Dashboard Grid:** Replaces the traditional boring linear blog timeline with dynamic grid modules that intelligently adapt across viewports.
- **Smart Aspect-Fit Gallery:** Employs explicit object-cropping (`object-fit: cover;`) constraints to align imagery smoothly across matching horizontal baselines.
- **Scroll Reveal Animations:** Powered by native JavaScript Intersection principles to gracefully glide and fade tiles upward as the reader scrolls.
- **Responsive Flex Design:** Built from the ground up to collapse cleanly into single-column layouts for optimal mobile viewing.

## 📂 Project Structure

```text
personal-blog/
├── index.html          # Main blog layout markup structure
├── style.css           # Custom styling rules, color variables, and grid definitions
├── script.js          # Intersection and viewport scroll reveal micro-logic
└── assets/             # Folder containing images (kimaya.jpg, hobbies.jpg, etc.)

---
```
Key features include customized grid-spanning blocks for featured projects, perfectly aspect-ratio-cropped gallery showcases utilizing smart CSS object-fit mechanics, and elegant viewport-tracking JavaScript fade-and-glide slide reveal animations. It provides a cohesive space to display technical implementations alongside creative hobbies and social media platforms, making it the perfect bridge between a clean professional portfolio and an expressive personal journal.

This project is open-source and free to use. Feel free to fork it, modify it, or use it to express your own digital creative journey!!
